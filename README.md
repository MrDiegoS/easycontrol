# Bem-vindo ao EasyControl!

Olá, eu sou o EasyControl, sua solução web para controlar suas dívidas de forma mais simples e rápida.


# Funções

- Cadastro de usuários
- Cadastro de parâmetros para calcular inadimplências
- Cadastro de Inadimplências
- Ambientes administrativo 

# Execução

	Como ao solução utiliza um banco SQL local, é necessário para o funcionamento do mesmo que a connectionStrings = ContextEasycontrol tenha o campo AttachDbFilename atualizado com o diretório onde a solução foi copiada
